# My Blog - 现代技术博客网站

一个模仿现代技术博客风格（如ibug.io）的个人博客网站，采用响应式设计和现代前端技术栈。

## 🎯 设计特色

### 现代简约设计
- **简洁色彩方案**：使用蓝黑色调为主的现代配色
- **充足留白**：专业的排版和良好的视觉层次
- **精美渐变**：首页背景采用渐变色增强视觉冲击力

### 响应式布局
- **移动优先**：从移动设备开始设计，逐步增强到桌面设备
- **断点优化**：768px和480px智能断点适配
- **灵活网格**：CSS Grid和Flexbox实现自适应布局

### 用户体验优化
- **流畅动画**：悬停效果和平滑过渡动画
- **移动端导航**：汉堡菜单和触摸友好的交互
- **博客阅读**：专门的博客详情页面，优化阅读体验

## 🛠️ 快速开始

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd my-personal-website
   ```

2. **本地预览**
   - 直接打开 `index.html` 文件
   - 或使用本地服务器：
     ```bash
     python -m http.server 8000
     # 访问 http://localhost:8000
     ```

3. **自定义内容**
   - 编辑 `script.js` 中的 `blogPosts` 数组添加文章
   - 修改 `index.html` 更新个人信息
   - 调整 `style.css` 自定义样式

## 📝 博客管理

### 添加新文章
编辑 `script.js` 中的 `blogPosts` 数组：

```javascript
{
    id: 5,
    title: "文章标题",
    excerpt: "文章摘要",
    content: `文章HTML内容`,
    date: "2024-12-01",
    category: "分类",
    readTime: "5 min"
}
```

### 支持的特性
- **富文本内容**：支持HTML标签和列表
- **阅读时间**：自动计算展示
- **分类标签**：按技术领域组织
- **发布日期**：时间线排序

## 🎨 自定义主题

### 颜色主题
修改CSS变量自定义主题：

```css
:root {
    --primary-color: #2563eb;  /* 主色调 */
    --text-primary: #1e293b;   /* 主文字色 */
    --bg-primary: #ffffff;     /* 背景色 */
}
```

### 字体设置
调整字体配置：

```css
:root {
    --font-family: 'Inter', sans-serif;
}
```
