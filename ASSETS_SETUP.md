# 天琴三颗星背景图片设置说明

## 📸 图片文件要求

请将天琴三颗星的两张照片放在 `assets/` 目录下：

### 文件命名及规格

**PC端横版背景** (推荐尺寸 1920x1080 或更大):
- 文件名：`lyra-stars-horizontal.jpg`
- 格式：JPEG 或 PNG
- 比例：16:9 宽屏比例

**移动端竖版背景** (推荐尺寸 1080x1920 或更大):
- 文件名：`lyra-stars-vertical.jpg`
- 格式：JPEG 或 PNG
- 比例：9:16 竖屏比例

## 🎨 图片优化建议

### 质量要求
- **高分辨率**: 确保图片清晰锐利
- **适当压缩**: 平衡文件大小和质量
- **曝光合适**: 避免过曝或过暗

### 构图建议
- **PC横版**: 天琴三颗星在水平方向分布均匀
- **移动竖版**: 天琴三颗星在垂直方向构图美观
- **主体突出**: 星体清晰，背景深邃

## 🔧 自定义背景大小

如果需要调整背景图片比例，可修改CSS中的尺寸设置：

```css
/* 在 style.css 中调整背景属性 */
body {
    background-size: cover; /* 或 contain */
    background-position: center;
}
```

## 📱 响应式调试

### PC端测试
- 浏览器缩放到宽屏查看横版背景效果
- 检查文字在星空背景上的可读性

### 移动端测试
- 使用浏览器开发者工具模拟移动设备
- 检查竖版背景在手机上的显示效果

## 🎯 替代方案

如果还没有合适的天琴三颗星照片，可以考虑：

   - 下载高质量的天琴座星系图片
   - 裁剪为所需尺寸

2. **使用AI生成**

3. **临时占位图片**
   - 暂时使用星空的通用图片
   - 后续替换为天琴三颗星特写
