# 改进版天琴三颗星背景设置说明

## 🎯 新的背景布局设计

现在天琴三颗星的图片使用方式已经优化，与您提供的网站示例保持一致：

### 📱 **设计特色**

**顶部横幅区域 (Hero Section)**
- **PC端**: 天琴三颗星横版图片作为背景
- **移动端**: 天琴三颗星竖版图片作为背景
- **透明渐变覆盖**: 半透明蓝色渐变层增强文字可读性

**整页背景**
- **固定背景**: 整个页面使用天琴三颗星图片作为深度背景
- **内容半透明**: 所有内容区域使用毛玻璃效果
- **响应式切换**: PC端横版背景，移动端竖版背景

## 📸 图片文件要求（保持不变）

### 文件名和规格
**PC端横版背景**:
- `assets/lyra-stars-horizontal.jpg` (建议 1920x1080)

**移动端竖版背景**:
- `assets/lyra-stars-vertical.jpg` (建议 1080x1920)

## 🎨 视觉效果特点

### 层次感设计
1. **底层背景**: 固定的星空调背景
2. **顶部横幅**: 吸引注意力的主横幅
3. **内容层**: 半透明毛玻璃效果的内容区块
4. **文字层**: 清晰可读的文本内容

### 响应式设计
- **PC端**:
  - 横幅使用横版图片，背景使用横版图片
  - 固定背景效果（视差滚动）

- **移动端**:
  - 横幅和背景都使用竖版图片
  - 滚动背景（更好的触摸体验）

## 🔧 自定义选项

### 调整透明度
```css
/* Hero区域渐变层 */
.hero::before {
    background: linear-gradient(135deg, rgba(38, 99, 235, 0.5) 0%, rgba(101, 31, 255, 0.5) 100%);
}

/* 内容区域背景 */
.section {
    background: rgba(255, 255, 255, 0.85);
}
```

### 修改渐变颜色
```css
.hero::before {
    background: linear-gradient(135deg, rgba(您的颜色, 0.5) 0%, rgba(您的颜色, 0.5) 100%);
}
```
